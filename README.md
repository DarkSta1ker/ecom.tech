### Для установки зависимостей

### `npm i`

### Для запуска проекта

### `npm start`

#### В ветке base лежит задание, выполненное по указаниям, нет ничего лишнего.
#### В ветке main лежит задание, выполненное с доработками: 
#### - фильтр по темам
#### - фильтр по цене
#### - обращение за ресурсами через api выполняется только после того, как пользователь перестал печатать
#### - пагинация (Пришлось усложнить запросы по api)
#### - ленивая загрузка
