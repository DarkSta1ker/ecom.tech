### Для просмотра работы на локальном компьютере необходимо скопировать репозиторий и выполнить:

### Для установки зависимостей

### `npm i`

### Для запуска проекта

### `npm start`

#### В ветке base лежит задание, выполненное по указаниям, нет ничего лишнего.
#### В ветке main лежит задание, выполненное с доработками(оно же залито на GHPages): 
#### - фильтр по категориям товаров
#### - фильтр по цене
#### - обращение за ресурсами через api выполняется только после того, как пользователь перестал печатать(debounce)
#### - пагинация (Пришлось усложнить запросы по api)
